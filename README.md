# Система навигации мобильного робота

## 📌 Описание проекта

Данный проект представляет собой программную систему управления мобильным роботом для задач складской логистики.  
Робот способен определять своё положение на карте, строить маршрут до заданной точки и автономно перемещаться, избегая препятствий.

Система реализует интеграцию алгоритмов локализации, планирования пути и управления движением.

---

## ⚙️ Основной функционал

### 📍 Локализация робота
- Определение позиции и ориентации робота на карте
- Использование данных лидара (сканирование окружающей среды)
- Уточнение положения в процессе движения

### 🗺️ Работа с картой
- Дискретная карта в виде сетки клеток
- Поддержка типов клеток:
  - препятствия
  - свободное пространство
  - точки интереса
  - стартовая позиция
  - "шоссе" (оптимизированные пути)

### 🧭 Планирование маршрута
- Алгоритм поиска пути **A\***
- Использование эвристики (манхэттенское расстояние)
- Поддержка ускоренного поиска через граф путевых точек (highway)

### 🤖 Управление движением
- Пошаговое движение по маршруту:
  - поворот к цели
  - движение вперёд
  - коррекция позиции
- Обработка ошибок движения
- Остановка при столкновении

### 📡 Работа с лидаром
- Приём данных по UDP
- Обработка массива расстояний
- Визуализация сканирования

### 🔌 Управление роботом
- Отправка команд движения по UDP
- Получение состояния робота в реальном времени
- Обработка телеметрии

### 🖥️ Графический интерфейс
- Отображение карты и пути
- Визуализация данных лидара
- Выбор целевой точки
- Запуск/остановка движения
- Логирование событий

## ▶️ Запуск проекта

1. Запустите симулятор (приложение-клиент) из папки **симулятор_роботов** - файл **IoTRobotWorld.exe**
2. Запустите приложение-сервер (управляющее приложение)
3. Перед запуском убедитесь, что порты и IP адреса в обоих приложениях указаны верно
4. Также убедитесь что порт лидаров в серверном и клиентском приложении совпадают
5. В симуляторе нажмите кнопку подключения, затем запустите симуляцию Show Simulator Viewport -> Start Simulation
6. В серверном приложении также нажмите кнопку подключения
7. Выберите точку интереса из списка и нажмите кнопку отправки робота в выбранную точку
8. Робот начнет движение

---

## 🧪 Порядок работы приложения

1. Робот определяется на карте с помощью лидара  
2. Пользователь выбирает точку интереса  
3. Строится маршрут  
4. Робот последовательно проходит клетки пути  
5. При движении позиция уточняется
